# Example Project OpenVZ
This is an example setup and configuration of an virtuozzo host over ansible.

## Requirements

### Vagrant
Vagrant Version: 2.0.2

```
vagrant plugin install ansible
vagrant plugin install vagrant-reload
```

## Usage

```
vagrant up
vagrant ssh
sudo vzlist 
```
